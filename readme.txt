This plugin extract all the outbound links to your Urlshortener.co Account and short these automatically.


== Description ==

This plugin extract all the outbound links of your Blog or Website to your <a href="http://www.urlshortener.co" title="Link Shortener">Urlshortener.co Account</a> and short these automatically with Anti-Adblock Solution System to earn money. This is ideal if you want a count of the amount of links on every page of your website, or want to earn money with these links. So It can be used for things: such as collecting all the internal links on your website and then creating a CSV file which Urlshortener can create in seconds.



== Installation ==

This section describes how to install the plugin and get it working.

1. Visit 'Plugins > Add New'
2. Upload UrlshortenerLinkExtractor.zip
3. Activate Urlshortener Link Extractor from your Plugins page.
4. Give your API Key and API Token in Settings of Plugin.
5. Show all you outbound links in your Urlshortener.co Account
